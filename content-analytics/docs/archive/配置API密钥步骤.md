# 配置API密钥步骤

## ✅ 代码已修复

我已经修复了聊天API的代码，现在它会正确使用用户设置中的API密钥了。

## 📝 配置步骤

### 第一步：登录系统
1. 访问 http://localhost:5173
2. 使用你的账号登录（如果没有账号，可以先注册）

### 第二步：进入设置页面
登录后，点击导航栏中的"设置"按钮，进入设置页面

### 第三步：配置API密钥
1. 在设置页面，找到"AI 服务配置"部分
2. 选择你想要使用的AI服务商（DeepSeek / OpenAI / iFlow）
3. 在对应的输入框中输入你的API密钥
4. 点击"保存设置"按钮

### 第四步：测试配置
1. 点击"测试API密钥"按钮验证密钥是否有效
2. 如果显示"API密钥配置正确"，说明配置成功

## 🔑 获取API密钥

### DeepSeek（推荐，价格便宜）
- 访问：https://platform.deepseek.com/
- 注册账号 → 进入控制台 → 获取API密钥
- 密钥格式：`sk-xxxxxxxxxxxxxxxx`

### OpenAI
- 访问：https://platform.openai.com/
- 注册账号 → API Keys → 创建新密钥
- 密钥格式：`sk-xxxxxxxxxxxxxxxx`

### iFlow
- 访问：https://iflow.cn/
- 注册账号后获取API密钥

## 🧪 测试配置

配置完成后，你可以：
1. 创建一个新的对话
2. 或打开一个分析结果的详情页，点击"与AI讨论"
3. 发送一条消息测试AI是否正常回复

## ⚠️ 注意事项

- API密钥是敏感信息，请妥善保管
- 使用AI服务可能会产生费用，请查看服务商的定价
- 如果遇到问题，检查网络连接和服务商服务状态
